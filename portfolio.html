Projects

Personal Blog Website

Designed and developed a responsive personal blog using HTML5, CSS, and a touch of JavaScript. The blog features a clean layout, categorized posts, and a comment section. Focused on semantic markup and accessibility, ensuring the site is easy to navigate for all users, including those using screen readers.

E-commerce Product Listing

Built a product listing page for a fictional online store. The page displays product names, descriptions, prices, and an "Add to Cart" button for each item. Implemented a "Sold Out" status with visual cues and used CSS to create a visually appealing, table-like layout without relying on advanced frameworks.

Responsive Photo Gallery

Created a mobile-friendly photo gallery that displays images in a grid. Used CSS media queries to adjust the layout: single column on mobile, two columns on tablets, and three columns on desktops. The gallery is easy to browse and loads quickly, with proper image alt text for accessibility.
